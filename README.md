The **Online Student Card Creation System (SCCS)** is a full-stack web application that bridges the gap between students and university administration by streamlining the student card request and issuance process.

Instead of physical queues and manual paperwork, students can submit their details, upload required documents, and receive a digital student card вҖ” all through a clean, intuitive web interface. Administrators gain a centralised dashboard to review, approve, and manage student records with full visibility and control.

> Built with scalability and real-world usability at its core, this system is designed to evolve alongside institutional needs.

**Step-by-step flow:**

1. **Register** вҖ” A student completes the online registration form with personal details.
2. **Upload** вҖ” Proof of registration and a profile photo are submitted via the upload portal.
3. **Review** вҖ” The system processes the submission automatically via API validation; admins retain override authority.
4. **Issuance** вҖ” Upon approval, a virtual student card is generated and linked to the student's account.
5. **Access** вҖ” The student logs in at any time to view their digital card.

## рҹҡҖ Getting Started

### Prerequisites

Ensure the following are installed on your local or server environment:

- **PHP** >= 7.4 or **Node.js** >= 16.x
- **MySQL** >= 8.0
- **Apache** or **Nginx** web server
- A modern web browser

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/sihledladla/student-card-system.git
cd student-card-system

# 2. Configure environment variables
cp .env.example .env
# Edit .env with your database credentials and API keys

# 3. Import the database schema
mysql -u root -p < database/schema.sql

# 4. Install dependencies (if using Node.js backend)
npm install

# 5. Start the development server
npm run dev
# OR for PHP: configure your Apache/Nginx vhost to point to /public
```

> вҡ пёҸ **Note:** Ensure your `.env` file is never committed to version control. It is already included in `.gitignore`.
